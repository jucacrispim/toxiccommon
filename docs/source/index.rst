Toxiccommon: Coordination and interfaces for toxicbuild
=======================================================

Toxiccommon contains stuf related to coordination and interfaces
for the other toxicbuild parts

Check the full toxicbuild documentation `here <https://toxicbuild.poraodojuca.dev>`_


Installation
------------

To install toxiccommon, use pip:

.. code-block:: sh

    $ pip install toxiccommon


And that's it!
